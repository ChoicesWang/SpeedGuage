# SpeedGuage
##  Interpolator 插值器 
  例如四驱汽车就是用差速锁来分配轮子的动力的。
  
## Android 动画一般都需要一个差速器(插值器)。
  常见的插值器包括：先快后慢，匀速，回弹，跳跃，等等
  然后也可以自己写插值器。一般的插值器都是一个一元二次方程。

## 这个Demo就是用来展示不同的 Android 动画插值器 具体的运动表现形式。

  你可以添加不同的差速器来观察他们的效果。
